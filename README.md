# IMPORTX

## Descripción del proyecto
IMPORTX es un aplicación móvil destinada a facilitar y optimizar la importación de vehículos.
Permite la gestión completa del stock de vehículos, el cálculo y control de importaciones reales, 
la administración de clientes y leads, así como la automatización de 
comunicaciones con clientes y vendedores.

La aplicación incluye módulos para: 
- Gestión de stock de vehículos.
- Cálculo y seguimiento de importaciones.
- Alta rápida de vehículos.
- Gestión de clientes y leads.
- Configuración del negocio y notficaciones automáticas.
